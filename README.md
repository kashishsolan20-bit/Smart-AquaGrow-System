# Smart-AquaGrow-System
Smart AquaGrow is an automated plant watering system that monitors soil moisture and controls water supply accordingly.It uses Arduino and soil moisture sensor for efficient water management.

# Introduction
This project presents an automated solution for watering plants efficiently by providing the right amount of water at the right time. The system works in three main stages:

1. Sensing the moisture level of the soil  
2. Determining whether the soil is dry or wet  
3. Controlling the water pump accordingly  

The soil moisture sensor continuously monitors the soil condition and sends data to the Arduino UNO. Based on this data, the Arduino controls the water pump by turning it ON or OFF.

This approach helps in:
- Preventing overwatering  
- Avoiding underwatering  
- Saving water  

---

## ⚙️ Working Principle
- The soil moisture sensor is placed in the root zone of the plant.  
- It measures the moisture level and sends data to the microcontroller (ATmega328 in Arduino UNO).  
- If the soil is dry → Pump turns ON  
- If the soil is wet → Pump turns OFF  

In advanced versions, GSM modules can be used to send updates to the user, but this project focuses on a simple and cost-effective household solution.
